# Профиль и настройки

## Зачем это нужно

Когда в команде 30 человек, и все упоминают друг друга через `@` и\
комментируют страницы — важно, чтобы ваше имя отображалось так, как\
вы хотите. Не как `alex.teroshkin@banank.eu`, а как «Алексей\
Терёшкин» или хотя бы «alex».

Раздел «Profile» — личные данные, которые видят коллеги,\
когда вы что-то делаете в документации.

## Где найти

Правый верхний угол → клик по аватару → **Profile** в выпадающем\
меню. Откроется страница `/settings/profile`.

## Что можно настроить

### Никнейм (Nickname)

Основное имя, которое отображается везде: в комментариях, в\
упоминаниях, в списке онлайн-пользователей, в авторстве правок,\
в уведомлениях.

Если ник не задан, система делает fallback по цепочке:\
**first name + last name → first name → nickname → email**.

### Имя (First name) и фамилия (Last name)

Опциональные поля. Полезны, если:

* Вы хотите, чтобы в документации вас видели по реальному имени\
  («Алексей Терёшкин» в Activity feed выглядит лучше, чем\
  `alex.teroshkin@gmail.com`).

* Вы работаете в компании, где принято обращаться по имени-фамилии\
  (бухгалтерия, ГОСТ-документация и т.п.).

Если заданы оба — первой строкой отображается `First Last`, а\
никнейм (если тоже задан) — как подпись через `@nickname·email`.

### Email (read-only)

Показан для справки. Менять email через этот экран нельзя —\
обращайтесь в поддержку (смена почты требует повторной верификации\
и связана с историей авторств).

### Avatar

Аватар подтягивается автоматически из источника, через который вы\
логинились:

* **GitHub login** — ваша GitHub-аватарка.

* **Email / OTP** — инициал имени на цветном фоне.

Загрузка собственной аватарки пока не поддерживается (в планах).

## Как сохранить

Измените поля → кнопка **Save changes** становится активной, если\
есть несохранённые изменения → клик. Через секунду появится\
зелёная плашка «Saved», новые данные сразу распространяются на всех\
коллег (для них ваш ник обновится в ближайшие минуты — когда у них\
обновится JWT-токен).

## Где отображается
| Место | Что показывается |
| -- | -- |
| Комментарий / упоминание | `First Last` если есть, иначе nickname, иначе email |
| Activity feed | Тоже resolveDisplayName |
| Online-аватары в правом углу страницы | Avatar + tooltip с именем |
| Notifications «X mentioned you» | «Alice Smith mentioned you» |
| Автор правки в History | То же имя что в комментариях |

Последовательная fallback-цепочка применяется везде, чтобы\
отображение было одинаковое. Если где-то видите свой email —\
скорее всего ни first/last, ни nickname не заданы; заполните в\
профиле.

## Пример

Допустим, у Алексея:

* email: `alex.teroshkin@gmail.com`

* nickname: `teroshkin`

* first_name: `Alexey`

* last_name: `Tereshkin`

В комментариях будет: **Alexey Tereshkin**, под ним — `@teroshkin · alex.teroshkin@gmail.com`.

Если убрать last_name, станет: **Alexey**, под ним — `@teroshkin · alex@...`.

Если убрать и first_name, и last_name, останется: **teroshkin**,\
без подписи.

## Тема оформления

В том же меню аватара — toggle **Dark Mode**. Мгновенно переключает\
тему интерфейса. Сохраняется в локальном хранилище браузера (на\
каждом устройстве — своя настройка).

## Выход из аккаунта

Меню аватара → **Log Out**. Выкидывает вас из всех вкладок,\
подтверждения не требуется. После логаута перенаправление на `/login`.

## Приватность

* Email виден другим участникам проектов только если вы сами его\
  указали в комментарии / странице. В системных местах (Activity\
  feed, mention) он скрыт.

* First / last name видны коллегам в проектах, к которым у них есть\
  доступ.

* Сторонние пользователи без общих проектов вашу информацию не\
  увидят (кроме публичных проектов, где упоминания делаются с\
  отображаемым именем).

## Ограничения

* Загрузить свою аватарку — пока нельзя.

* Изменить email — через поддержку.

* Никнейм должен быть непустым; лимиты — до 256 символов.

* First / last name — до 128 символов каждый; UTF-8, любые языки.

* Удалить аккаунт — через поддержку (GDPR-compliance, удаляем все\
  данные или анонимизируем по запросу).

## Типовые сценарии

**Первый раз зашёл.** Сразу поставьте ник и имя-фамилию. Потом в\
комментариях коллеги увидят нормальное имя вместо email.

**Перешёл из другой компании.** Email старый, профиль надо\
обновить. Зайдите в Profile, замените ник / имя на актуальные.

**Международная команда.** First / last — на латинице, чтобы все\
коллеги могли прочитать ваше имя.
